# IOTA ZMQ Client

This package provides a React Web UI to show the consumption of an IRI node zmq stream passed to the browser using websockets.

## Pre-requisites

```shell
npm install
```

## Development

This will run the Web UI at <http://localhost:3001>

```shell
npm run start
```
